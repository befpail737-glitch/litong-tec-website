## 部署指南

### 部署步骤

1. **备份现有网站**
   - 备份当前首页
   - 备份数据库（如有）

2. **上传文件**
   ```bash
   1. robots.txt → 根目录
   2. sitemap.xml → 根目录
   3. optimized-homepage.html → 重命名为 index.html
   4. pages/ 目录 → 根目录
   ```

3. **验证部署**
   - 检查页面加载
   - 验证SEO元素
   - 测试内部链接

4. **提交搜索引擎**
   - Google Search Console
   - Bing Webmaster Tools

详细步骤请查看: [DEPLOYMENT-CHECKLIST.md](DEPLOYMENT-CHECKLIST.md)
