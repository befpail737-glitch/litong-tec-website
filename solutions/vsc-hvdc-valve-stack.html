VSC-HVDC Valve Stack Solution

By LiTong HVDC Solutions Team | Solution Type: Power Grid | Application: Ultra-High Voltage Transmission

Our complete VSC-HVDC (Voltage Source Converter High-Voltage Direct Current) valve stack solution leverages CRRC Times Electric's industry-leading 6500V T-IGBT-G6500 modules to deliver exceptional performance, reliability, and efficiency for ultra-high voltage power transmission systems. This solution enables grid operators to achieve >99% transmission efficiency while maintaining superior power quality and control.

Solution Overview

The VSC-HVDC valve stack solution is designed for long-distance, high-capacity power transmission with minimal losses. Key features include:

  • Ultra-High Voltage Capability: Supports 6500V DC transmission systems for maximum power transfer
  • Exceptional Efficiency: Achieves >99% conversion efficiency with advanced modulation techniques
  • Grid Stability: Provides independent control of active and reactive power for enhanced grid stability
  • Black Start Capability: Can energize passive networks without external power sources

Core Components

The solution is built around CRRC Times Electric's advanced power semiconductor portfolio:

T-IGBT-G6500 Ultra-High Voltage Module

The cornerstone of the solution, the T-IGBT-G6500 delivers uncompromising performance in demanding HVDC applications:

  • 6500V blocking voltage with 750A continuous current capability
  • Industry-leading RGA (Reverse Gate Avalanche) protection for enhanced ruggedness
  • Press-pack configuration with spring-loaded pressure contact for reliable assembly
  • Qualified to stringent grid standards including IEC 62577 and IEEE 1547

Complementary Components

  • Gate Drive Units: High-isolation gate drivers optimized for T-IGBT-G6500 switching characteristics
  • DC-link Capacitors: Low-inductance, high-voltage capacitors for stable DC bus voltage
  • Voltage Sensors: Precision high-voltage dividers for accurate voltage measurement
  • Control System: Advanced DSP-based controller with real-time power control algorithms

System Architecture

The VSC-HVDC valve stack system architecture is optimized for grid applications:

  • Modular Design: Submodules arranged in series for scalable voltage levels
  • Redundancy: N+1 redundancy configuration for improved system availability
  • Cooling System: Advanced liquid cooling with redundant pumps and monitoring
  • Protection Features: Comprehensive fault detection and protection algorithms

Performance Benefits

Customers implementing our VSC-HVDC valve stack solution report significant improvements:

  • Transmission Efficiency: 99.2% efficiency achieved in field deployments
  • Reliability: 99.9% system availability with proven module reliability
  • Power Quality: Superior harmonic performance with <5% THD
  • Control Precision: Millisecond-level response for dynamic grid support

Application Examples

Our solution has been successfully deployed in diverse grid applications:

  • Offshore Wind Integration: Connecting remote offshore wind farms to mainland grids
  • Intercontinental Links: Cross-border power exchange between neighboring countries
  • Urban Power Supply: Underground HVDC links for city center power delivery
  • Renewable Integration: Large-scale solar and wind farm grid connections
